# Pixel-Perfect-Aligner
A GIMP 3 plugin that converts messy, AI-generated or upscaled sprites into clean, grid-aligned pixel art by **re-sampling** the selected area into a target pixel grid (e.g. 64Ã—64), with optional **noise cleanup**, **palette reduction**, **binary alpha**, **outline generation**, and **alpha-bleed fix** to prevent dark halos in-game.
